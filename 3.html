3.Cabeçalhos

A tag title define o título da página

<title>

Título da página

</title>

A tag link serve para inserir um arquivo externo dentro do html(css)

<link/>

A tag script serve para trazer um arquivo externo(javascript) para executar no navegador

<script>

pode-se inserir o código aqui mesmo ou "puxar" de outro lugar da internet

</script>

A tag style serve para escrever o código css dentro do próprio html

<style>


Aqui vai o código de css(folhas de estilo)

</style>

A tag meta serve para mudar todos os dados internos da página.
São meta-dados e eu defino esses dados internos sobre o meu site.

<meta/>

  • Quem criou a página
  • Descrição do site.
  • Quando o criou
  • Fazer com que o site atualize a cada 30 segundos,30 minutos, 1 hora,etc...

    • Assim,aparecerá no google o título dela (no google).
    Essa tag basicamente irá referenciar informações do meu site,ou seja, palavras-chave.

A tag base especifica qual é o endereço original do meu site(pouco usada).
Porém, é sempre bom usar todas as tags disponíveis para especificar todos os detalhes sobre o meu site.

<base/>

exemplo:

<base href="http://www.meusite.com.br/">
