6.Listas não ordenadas

Inserindo listas não ordenadas numa página:

A Sintaxe do código é essa
<ul>
<li>Aqui vai o "Item 1". </li>
</ul>
Depois cria-se outro <li> </li>
e assim por diante ...

Ingredientes para um bolo(Exemplo)

  • Farinha
  • Açúcar
  • Fermento
  • Ovos
  • Manteiga

Estilizando a lista não ordenada:


1.None(sem nada)


    <ul style="list-style-type:none">
    <li>
  • Item 1
    • </li>
    </ul>

1.Disc(Bolinha preta)


    <ul style="list-style-type:disc">
    <li>
  • Item 1
    • </li>
    </ul>

3.Square(Quadrado preto)


    <ul style="list-style-type:square>
    <li>
  • Item 1
    • </li>
    </ul>

4.Circle(Bolinha branca)


    <ul style="list-style-type:circle>
    <li>
  • Item 1
    • </li>
    </ul>
