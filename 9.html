9-Links

Como inserir links dentro de um documento HTML


Antes inserir a tag <a> </a>
conheça alguns atributos e para que servem:

atributo href:

Serve para pegar o link que se refere ao local ou documento que se deseja acessar.

<a href="index.html"> Clique aqui para acessar a página principal(index.html) </a>

atributo download:

Serve para quando clicar no link, ele não vai abrir o documento ou pdf, ele vai baixar para a máquina do usuário.

<a href="documento.pdf" download> Baixar documento em PDF </a>

atributo target:

Serve para quando clicar no link, irá abrir ele em uma nova aba do navegador do usuário.

<a href="index.html" target="_blank"> Abrir pdf em nova aba.(mas poderia ser uma página também) </a>
