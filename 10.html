10-Tabelas

Como inserir tabelas dentro de um documento HTML


Antes inserir a tag <table> </table>
conheça algumas subtags, atributos e para que servem:

Subtag tr:

Serve para definir o número de linhas que a tabela tem
A cada tag <tr></tr> aberta é mais uma linha que a tabela terá

<table>
	<tr> <\tr>
<table>


Tag td:

Serve para definir o número de colunas que a tabela tem, fica dentro da tag tr e podem existir várias colunas dentro de uma linha.
A cada tag <tr></tr>aberta, é mais uma linha que a tabela terá

<table>
	<tr> 
			<td></td>
	<\tr>
<table>

atributo width:

Serve para definir o tamanho da tabela em largura

<table width="200"</table>
