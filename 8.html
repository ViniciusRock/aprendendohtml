8-Imagens

Como inserir uma imagem dentro de um documento HTML


Antes inserir a tag img, conheça alguns atributos e para que servem:

atributo src:

Serve para especificar o local onde a imagem está localizada,seja numa pasta anterior ( ../pasta/imagem.png,
na mesma pasta do arquivo html ou em um servidor externo).

<img src="html.png"/>


atributo height:

Serve para especificar a altura da foto(Height = Altura).
*DICA* -> Se definir só a altura ou só a largura, a foto auto redimensiona para ficar nas proporções corretas.

<img src="html.png heitgh="100"/>

atributo width:

Serve para definir a largura da foto(width = Largura).
*DICA* -> Se definir só a altura ou só a largura, a foto auto redimensiona para ficar nas proporções corretas.

<img src="html.png width="100"/>


atributo title:

Muito usado. Ao passar o mouse em cima, mostra um texto dizendo algo sobre a foto, seja o nome, a legenda, ou descrição dela.
É definido pelo programador.
<img src="html.png" height="100" title="Imagem html5"/>

atributo alt:

Pouco usado, mas não menos importante.
É uma descrição da imagem quando ela não carrega a tempo.
Ao passar o mouse em cima, nesse caso aparecerá uma descrição da foto feita pelo programador da mesma forma que o title.
Só que nesse caso somente quando não estiver totalmente carregada.

<img src="html.png"
alt="descrição quando não carrega a imagem a tempo"/>

Exemplo de imagem com o atributo src="html.png", atributo height="200", width="200",
title="imagem html" e alt="imagem mostrando html" :

<img src="html.png" height="200" width="200" title="imagem html" alt="imagem monstrando html"/>
